# Front-End Application with React, Bootstrap, and API Integration

## Overview

This project is a front-end application built using React.js and styled with Bootstrap. The application interacts with an external API to perform full CRUD (Create, Read, Update, Delete) operations within a table interface. Additionally, it includes login and registration pages for user authentication.

## Features

- **CRUD Operations**: The application supports full CRUD functionality using a chosen public API.
  - **Create**: Add new items to the table.
  - **Read**: Display a list of items fetched from the API in a tabular format.
  - **Update**: Edit existing items directly within the table.
  - **Delete**: Remove items from the table.
- **User Authentication**: 
  - **Login**: Secure login page for user authentication.
  - **Registration**: Registration page for new users.
- **Responsive Design**: Fully responsive, ensuring usability across different devices and screen sizes.

## Technologies Used

- **React.js**: Front-end framework for building the user interface.
- **Bootstrap**: CSS framework for responsive and styled components.
- **External API**: Utilized for CRUD operations and data management.
